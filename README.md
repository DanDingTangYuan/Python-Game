# Python-Game
> An easy game project for me to do some technical test.

> 一個簡單的遊戲專案，用來應付期末報告

## img folder
> The folder of Image

> 是整個程式的圖片檔案目錄

## sound folder
> The folder of sound

> 是整個程式的音效檔案目錄

## res folder
> The fold for multimedia

> 程式的圖示、圖片、ui等目錄

## requirements
> The API that we need from outside

> 用於存放整個程式依賴的外部Python包列表
